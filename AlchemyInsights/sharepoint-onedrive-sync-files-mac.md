# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Installer installationsprogrammet, og Synkroniser OneDrive eller SharePoint filer på Mac 

Installer, installation, og Synkroniser OneDrive eller SharePoint filer på en Mac med vejledning fra [synkronisere filer med OneDrive synkronisering klient på Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Hvis OneDrive-app er installeret og installation på en Mac, men brugere oplever problemer med synkroniseringen, kan du se [rettelse sync problemer i forbindelse med OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&amp;rs=en-US&amp;ad=US).

> [!NOTE]
> Funktionen kendte flytte mappen i Windows OneDrive synkronisering er ikke tilgængelig til Mac.
